Create a page about yourself, and make sure to fulfill these criterias:

  1. Name your file index.html
  2. Make sure your page has a <title>
  3. Create a heading with the <h1> tag
  4. Change the color of your <h1> tag
  5. Change the background color (Hint: The selector for this is body and the declaration is background-color -- W3 Reference)
  6. Embed/insert min. 1 image
  7. Link your favorite website to your page
  8. Create an unordered list about things you like
  9. Optional: Add anything you want to make the page interesting :)
